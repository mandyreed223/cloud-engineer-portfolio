## About This Project

This project serves as my personal cloud portfolio website and provides an overview of my professional background, technical skills, cloud projects, and learning journey.

The website was built using HTML and CSS and deployed using Amazon S3 Static Website Hosting. It is designed to highlight the cloud technologies I have worked with and the projects I have completed while transitioning into Cloud Engineering.
